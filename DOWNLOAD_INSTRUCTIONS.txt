INSTRUCTIONS FOR DOWNLOADING SATELLITE DATA
============================================

Since the Google Drive integration currently doesn't support downloading 
non-Google Docs files (like TIF/GeoTIFF images), you have two options:

OPTION 1: Download Manually
----------------------------
1. Visit the Google Drive folders using the URLs below
2. Download all image files to your local machine
3. Upload them to this environment

Sentinel-2 URL: https://drive.google.com/drive/folders/15yN_kbDv615DukoMgJXPAVWreA5F-RwT
Sentinel-5P URL: https://drive.google.com/drive/folders/1-X9de6Fkqw-2NNo3sSc2ObqTQ-yVzt8D

OPTION 2: Use Google Drive API with Service Account (Recommended)
------------------------------------------------------------------
This requires setting up Google Drive API credentials.

OPTION 3: Use gdown or similar tools
-------------------------------------
Install: pip install gdown
Then use folder IDs to download entire folders.
